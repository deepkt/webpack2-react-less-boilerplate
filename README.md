# Webpack2 with React Boilerplate with Less, ESLint and HMR

### Installation

Requires [Node.js](https://nodejs.org/) v6+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd webpack2-react-less-boilerplate
$ npm install
$ npm run
```
